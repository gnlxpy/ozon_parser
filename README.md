# Ozon Parser (API)

## Описание
Тестовый проект собирающий данные через API с помощью асинхронной библиотеки httpx и bs4.

## Структура проекта
```
ozon_parser/
|-- main.py                # содержит основную запускаемую функцию
|-- common/                # вспомогательные функции по проверке/обработке данных
|-- requests_handler.py/   # обработчик запросов httpx
|-- errors.py/             # кастомные типы ошибок
```

## Файл с конфигом
Необходим файл .env со следующими переменными (используется dotenv для работы с переменными окружения):
```
COOKIE='{{ваша строка Cookie из любого запроса на ресурсе}}'
USER_AGENT='{{ваш User-Agent из Headers}}'
POSTFIX_URL_API='/api/entrypoint-api.bx/page/json/v2
```

POSTFIX_URL_API - окончание url api, полная ссылка выглядит так:
https://www.ozon.ru/api/entrypoint-api.bx/page/json/v2

Необходима только часть после домена (эта ссылка может измениться со временем)
